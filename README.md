# Wifi_Button_Enclosure
3D printable enclosure for a ESP-01 module based WiFi button with a 0.96" OLED display
Use the FreeCad file to modify the design or just download the STL file and print it.
The lid and box is snap fit so no need for screws!!

This enclosure is designed to fit a 3cm X 7cm PCB (double sided zero PCBs are pretty popular).<\br>
There are window cut-outs for:<\br>
1. 0.96" OLED display
2. 12x12mm tactile momentary push buttons with the plastic caps.
3. USB-C charding port
4. Power switch
5. Hole for adding a keychain loop
